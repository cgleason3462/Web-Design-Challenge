Summary of the Project

The purpose of this project was to analyze how the weather changes as you get closer to the equator. To accomplish this analysis, we first oulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather vd. latitude. Factors we looked at include: temperature, cloudiness, wind speed, and humidity. Tjis site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
